# Nebula Sports webOS

Homebrew Channel custom repository:

```text
https://raw.githubusercontent.com/retrocodes12/NebulaSports/main/repo.json
```

## Install

1. Open Homebrew Channel on webOS.
2. Open repository settings.
3. Add the custom repository URL above.
4. Refresh repositories.
5. Install **Nebula Sports**.

Direct IPK:

```text
https://raw.githubusercontent.com/retrocodes12/NebulaSports/main/com.nebulastreams.sports_1.0.13_all.ipk
```

If an older build is already installed and update fails, uninstall `com.nebulastreams.sports`, then install again.
